## Supabase Dashboard 접속

1. [Supabase Dashboard](https://supabase.com/dashboard)에 접속하여 로그인
2. 프로젝트 선택
3. 좌측 메뉴에서 **Table Editor** 클릭
4. `products` 테이블 선택

## 상품 등록 방법

### 단계별 가이드

1. **"Insert row" 버튼 클릭**

   - 테이블 상단 또는 하단의 "Insert row" 버튼을 클릭합니다.

2. **필수 필드 입력**

   - 아래 [필수 필드 설명](#필수-필드-설명) 섹션을 참고하여 필수 필드를 입력합니다.

3. **선택 필드 입력 (선택사항)**

   - `description`: 상품 설명을 입력합니다.

4. **저장**
   - "Save" 버튼을 클릭하여 상품을 등록합니다.

## 필수 필드 설명

### `name` (TEXT, 필수)

- **설명**: 상품명
- **예시**: "프리미엄 샤워 헤드", "스마트 수도꼭지"
- **주의**: 빈 값이면 안 됩니다.

### `price` (DECIMAL, 필수)

- **설명**: 상품 가격
- **형식**: 소수점 2자리까지 (예: `89000.00`)
- **예시**:
  - 89,000원 → `89000.00`
  - 159,000원 → `159000.00`
- **주의**: 0 이상의 값이어야 합니다.

### `category` (TEXT, 선택)

- **설명**: 상품 카테고리
- **가능한 값**: `shower`, `bath`, `sink`, `accessories`
- **예시**: `shower`
- **주의**: 정확한 값만 사용 (대소문자 구분)

### `stock_quantity` (INTEGER, 기본값: 0)

- **설명**: 재고 수량
- **예시**: `150`, `80`, `200`
- **주의**: 0 이상의 값이어야 합니다.

### `is_active` (BOOLEAN, 기본값: true)

- **설명**: 상품 활성화 여부
- **값**: `true` (활성화) 또는 `false` (비활성화)
- **주의**: `false`로 설정하면 상품 목록에 표시되지 않습니다.

### `description` (TEXT, 선택)

- **설명**: 상품 설명
- **예시**: "고음질 노이즈 캔슬링 기능, 30시간 재생"
- **주의**: 선택 필드이므로 비워둘 수 있습니다.

## 카테고리 값 매핑

| 카테고리 값   | 표시명   | 설명                           |
| ------------- | -------- | ------------------------------ |
| `shower`      | 샤워용품 | 샤워기, 샤워 헤드, 샤워 커튼   |
| `bath`        | 욕조용품 | 욕조, 목욕 가구, 욕조 액세서리 |
| `sink`        | 세면대   | 세면대, 수도꼭지, 거울         |
| `accessories` | 액세서리 | 수건걸이, 선반, 수납함         |

**중요**: 카테고리 값은 정확히 입력해야 합니다. 대소문자를 구분하므로 `Shower`나 `SHOWER`는 작동하지 않습니다.

## 샘플 데이터 예시

### 샤워용품 예시

```json
{
  "name": "프리미엄 샤워 헤드",
  "description": "고압 분사로 깨끗한 세정력, 물 절약 기능",
  "price": 89000.0,
  "category": "shower",
  "stock_quantity": 150,
  "is_active": true
}
```

### 욕조용품 예시

```json
{
  "name": "욕조용 목베개",
  "description": "편안한 목 지지, 방수 소재",
  "price": 39000.0,
  "category": "bath",
  "stock_quantity": 80,
  "is_active": true
}
```

### 세면대 예시

```json
{
  "name": "스마트 수도꼭지",
  "description": "터치 방식, 온도 조절 기능",
  "price": 159000.0,
  "category": "sink",
  "stock_quantity": 50,
  "is_active": true
}
```

### 액세서리 예시

```json
{
  "name": "벽걸이 수납 선반",
  "description": "욕실 공간 활용, 방수 코팅",
  "price": 49000.0,
  "category": "accessories",
  "stock_quantity": 200,
  "is_active": true
}
```

## 주의사항

### 1. `is_active` 설정

- `is_active`를 `false`로 설정하면 상품 목록에 표시되지 않습니다.
- 상품을 일시적으로 숨기고 싶을 때 사용합니다.
- 완전히 삭제하지 않고 비활성화하는 것이 좋습니다.

### 2. 가격 형식

- 가격은 반드시 소수점 2자리까지 입력해야 합니다.
- 예: `89000.00` (O), `89000` (X)
- 정수 값도 `89000.00` 형식으로 입력하세요.

### 3. 카테고리 값

- 카테고리 값은 정확히 입력해야 합니다.
- 대소문자를 구분하므로 `shower`만 허용됩니다.
- 잘못된 카테고리 값은 필터링에서 제외됩니다.

### 4. 재고 수량

- 재고 수량이 0이면 "품절"로 표시됩니다.
- 재고 수량이 10 미만이면 "재고 부족"으로 표시됩니다.
- 음수 값은 입력할 수 없습니다.

### 5. 자동 생성 필드

다음 필드들은 자동으로 생성되므로 입력하지 않아도 됩니다:

- `id`: UUID (자동 생성)
- `created_at`: 생성일시 (자동 생성)
- `updated_at`: 수정일시 (자동 생성)

### 6. 상품 수정

- 기존 상품을 수정하려면 테이블에서 해당 행을 클릭하여 수정합니다.
- `updated_at` 필드는 자동으로 업데이트됩니다.

### 7. 상품 삭제

- 상품을 삭제하려면 행을 선택하고 "Delete" 버튼을 클릭합니다.
- **주의**: 삭제된 상품은 복구할 수 없습니다.
- 대신 `is_active`를 `false`로 설정하는 것을 권장합니다.

## 문제 해결

### 상품이 목록에 표시되지 않을 때

1. `is_active`가 `true`인지 확인
2. 카테고리 값이 정확한지 확인
3. 브라우저 캐시를 지우고 새로고침

### 가격이 올바르게 표시되지 않을 때

1. 가격 형식이 `89000.00` 형식인지 확인
2. 소수점 2자리까지 입력했는지 확인

### 카테고리 필터가 작동하지 않을 때

1. 카테고리 값이 정확한지 확인 (`shower`, `bath`, `sink`, `accessories`)
2. 대소문자를 정확히 입력했는지 확인
